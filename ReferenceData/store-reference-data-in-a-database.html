This is a sample Reference Data Entity. For a real entity, some explanations of it use would be noted here.

To create a reference data hazu, you need this template. Copy it to the desired location, then modify the following three parameters:

  • Set the data-hazu-redirect-url to the URL (not the ID!) of the hazu directory. 
  • Copy the ID (not the URL!) of the hazu directory to the value attribute of hidden hazuid input field. 
  • Create an invite in the Sharing Menu, and copy the code (not the URL!) to the value attribute of hidden auth input field. 

You can delete this text once you are done but keep the form data for the update button.
